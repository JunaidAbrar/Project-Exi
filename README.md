Project EXi is a scalable, multi-tenant AI SaaS prototype designed to automate real estate rental operations, specifically focusing on rental matching, lead capture, and inspection booking. Containerized with Docker for streamlined deployment, the platform's architecture leverages FastAPI, LlamaIndex, Postgres, and Qdrant for managing high-performance vector embeddings, while utilizing the Groq API for its current demo phase. To efficiently manage and isolate property and lead data across multiple real estate agencies without feature creep, the system integrates robust Google Sheets synchronization to keep agent dashboards and records seamlessly up to date.
